La caja perfecta de herramientas!

Dataprep: Es un servicio de datos inteligente que utilizamos para examinar, limpiar y preparar datos estructurados y no estructurados de forma visual para procesos de análisis.

BigQuery: Es un almacén de datos empresariales que permite consultas de SQL de alta velocidad mediante el poder de procesamiento de la infraestructura de Google.

Datalab: Es una herramienta interactiva y pontente, creada para explorar, analizar, transformar y visualizar datos, así como compilar modelos de aprendizaje automático.

DataStudio: Es una herramienta de visualización de datos y creación de reportes, la cual incluye conectores para las principales herramientas de Google (Analytics, BigQuery, Sheets, etc).
